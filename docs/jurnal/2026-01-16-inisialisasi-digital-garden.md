# 16 Januari 2026: Inisialisasi Digital Garden

Hari ini, saya memulai proyek pertama saya dengan menggunakan **Antigravity Code Agent**. Kesan pertama saya terasa sangat mirip dengan VS Code—ekstensi dan tampilannya persis sama—sehingga saya merasa tidak asing dalam menggunakannya.

### Rencana Konten
Proyek yang saya garap kali pertama ini adalah *Digital Garden*, di mana saya berencana menjadikannya sebagai wadah untuk menampung:

1. **Jurnal Pribadi:** Untuk mencatat pengalaman dalam menyelesaikan masalah, proyek, atau eksperimen tertentu.
2. **Wiki Teknis:** Tempat menyimpan solusi masalah (*troubleshooting*) berdasarkan pengalaman pribadi.

Tulisan ini adalah langkah awal saya dalam menuangkan pengalaman pribadi ke dalam bentuk tulisan. Saya berharap wadah digital ini bisa mengabadikan semua catatan saya kedepannya agar tidak hilang begitu saja.
